## 🚀 Overview
StayFinder is a modern Airbnb-inspired rental platform built with Angular 20 and Supabase.

It allows users to:
- Browse properties
- View detailed listings
- Request bookings
- Manage properties via a host dashboard

---

## 🏗️ Architecture

```
Frontend (Angular 20)
    ↓
Supabase API (Auth, DB, Storage)
    ↓
PostgreSQL Database
```

---

## ✨ Features

### 🏠 Property Discovery
- Property listing grid
- Carousel sections
- City search

### 📄 Property Detail
- Image + description
- Availability calendar
- Booking form

### 📅 Booking System
- Date selection
- Price calculation
- Availability blocking

### 🧑‍💼 Host Dashboard
- Add properties
